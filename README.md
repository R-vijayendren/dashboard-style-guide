# Dashboard Style Guide

This React App describe the `Dashboard Style Guide`. Which means the dashbard can be designed in the way like this teach stack.

We have used React JS, SCSS/SASS.
Doesn't add too much of packages.

## Dependency Packages

- node-sass
- react-icons
